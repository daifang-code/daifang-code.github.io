缺陷描述

方法调用System.exit(...)

漏洞与风险

调用System.exit会关闭整个Java虚拟机。只在适当的时候才能关闭虚拟机。此类调用使这段代码很难被其它代码调用。 考虑用RuntimeException来代替。
