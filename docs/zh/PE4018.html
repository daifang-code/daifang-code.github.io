相同的分支

缺陷描述

代码无论条件语句和表达式如何，都执行相同的代码。这样的重复代码意味着该条件是不必要的（或可以合并多个条件），或者该代码不应相同（因此可能是复制粘贴错误）。

该缺陷主要以下面几种方式出现： 在then和else分支中具有相同代码的简单if-then-else语句。 链中连续分支中具有相同代码的任何其他if链 三元表达式，例如cond?expr1:expr2，其中expr1与expr2相同。 在不同case语句中具有相同代码的任何switch语句。

漏洞利用风险

低

Potential Mitigations

删除相同的分支的代码。

    if (x > 0) {
        return y;

    return y;

int test_tp2(int x, int y) {
    if (x > 0) {
        return y+2;
    }
    return 2+y;
}


int test_tp3(int x, int y) {
    int z,l;
    if (x > 0) {
        z = y+2;
        l = y-1;
        goto label4;
    }
    z = y+2;
    l = y-1;
label4:
    return z + l;
}
