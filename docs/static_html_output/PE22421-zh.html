PE22421: E713

Test for membership should be 'not in'

缺陷信息

缺陷ID
PE22421
标签
flake8
目录
Code style and efficiency
作者
pinpoint

成员资格测试应使用x not in the_list而不是not x in the_list。前一个例子更简单可读。

示例错误片段
my_list = [1, 2, 3]
if not num in my_list:
    print(num)
最佳修复
my_list = [1, 2, 3]
if num not in my_list:
    print(num)
