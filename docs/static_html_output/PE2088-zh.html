禁止使用看起来像除法的正则表达式 (no-div-regex)

要求正则表达式字面量避开除法操作符。

function bar() { return /=foo/; }

缺陷详因

该规则用来消除除法操作符的歧义。

错误 代码示例：

/*eslint no-div-regex: "error"*/

function bar() { return /=foo/; }

正确 代码示例：

/*eslint no-div-regex: "error"*/

function bar() { return /\=foo/; }
