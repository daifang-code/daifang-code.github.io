SC2109: Suspicious condition

Instead of [ a || b ], use [ a ] || [ b ].

Instead of [ a || b ], use [ a ] || [ b ].

Problematic code:
[ "$1" = "-v" || "$1" = "-help" ]
Correct code:
[ "$1" = "-v" ] || [ "$1" = "-help" ]
Rationale:

|| can not be used in a [ .. ] test expression. Instead, make two [ .. ] expressions and put the || between them.

Exceptions:

None.
