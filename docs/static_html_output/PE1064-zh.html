PE1064: 内存不完整赋值

不正确地使用memset,memcpy或memmove函数,导致buffer无法被完全赋值。

缺陷信息

缺陷ID
PE1064
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

不正确地使用memset,memcpy或memmove函数,导致buffer无法被完全赋值。

漏洞与风险

缓存区大小计算不正确

不正确地使用memset,memcpy或memmove函数,会使得buffer的内存地址溢出。

缓解与预防

正确使用memset,memcpy或memmove函数。

漏洞利用威胁

中

演示示例

漏洞缺陷示例
void bad()
{
  int a[5];
  memset(a, 0, 5);//detect
}
漏洞缺陷修复示例
void good()
{
  int a[5];
  memset(a, 0, 5*sizeof(int));
}
