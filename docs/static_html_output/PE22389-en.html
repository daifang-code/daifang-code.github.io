PE22389: E223

Tab before operator

Bug info

BugTypeID
PE22389
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be only one space before or after operators.

Anti-pattern

Note: The → character represents a tab.

if x→in [1, 2, 3]:
    print(x)
Best practice
if x in [1, 2, 3]:
    print(x)
