PE22393: E231

Missing whitespace after ',', ';', or ':'

Bug info

BugTypeID
PE22393
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be whitespace after the characters ,, ;, and :.

Anti-pattern
my_tuple = 1,2,3
Best practice
my_tuple = 1, 2, 3
