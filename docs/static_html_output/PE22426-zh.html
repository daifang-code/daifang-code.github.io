PE22426: E741

Do not use variables named 'I', 'O', or 'l'

缺陷信息

缺陷ID
PE22426
标签
flake8
目录
Code style and efficiency
作者
pinpoint

名为I可以容易混淆。

将这些变量的名称更改为更具描述性的名称。

示例错误片段

本例中的代码可能会被误解为0 * 1.08（0乘以1.8）。

O = 100.0
total = O * 1.08
最佳修复

这个例子说明我们要乘以一个order变量另一个号码。

order = 100.0
total = order * 1.08
