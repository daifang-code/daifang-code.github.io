缺陷描述

方法调用低效的new String(String)构造函数会浪费内存空间

漏洞与风险

使用java.lang.String(String)构造函数会浪费内存，因为这样构造的对象在功能上与作为参数传递的String无法区分。这种情况下只需直接使用参数String。
