PE22413: E401

Multiple imports on one line

Bug info

BugTypeID
PE22413
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Imports from different modules should occur on different lines.

Anti-pattern
import collections, os, sys
Best practice
import collections
import os
import sys
