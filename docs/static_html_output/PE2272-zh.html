该规则已被ESLint v2.0移除，且由keyword-spacing规则替代。

(fixable) The --fix option on the command line automatically fixed problems reported by this rule.

Require spaces following return, throw, and case.

缺陷详因

Examples of incorrect code for this rule:

/*eslint space-return-throw-case: "error"*/

throw{a:0}

function f(){ return-a; }

switch(a){ case'a': break; }

Examples of correct code for this rule:

/*eslint space-return-throw-case: "error"*/

throw {a: 0};

function f(){ return -a; }

switch(a){ case 'a': break; }

Eslint版本

This rule was introduced in ESLint 0.1.4 and removed in 2.0.0-beta.3.
