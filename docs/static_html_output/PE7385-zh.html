缺陷描述

测试与NaN是否相等

漏洞与风险

此代码检查浮点值是否等于特殊的NaN（例如，if (x == Double.NaN)）。但是，由于NaN的特殊语义，没有值等于NaN，包括NaN本身。因此，x == Double.NaN 总是false的。 要检查x 中包含的值是否为特殊的NaN，请使用Double.isNaN(x) （如果x 是浮点精度，则使用Float.isNaN(x)）。
