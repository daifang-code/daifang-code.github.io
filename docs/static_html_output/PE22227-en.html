DUO117

This linter looks for use of the dl module.

From the documentation:

The dl module bypasses the Python type system and error handling. If used incorrectly it may cause segmentation faults, crashes or other incorrect behaviour.

Problematic code

import dl

Correct code

None - use the ctypes module instead

Rationale

Segmentation faults or crashes can lead to insecure behavior so we should avoid this module.

Exceptions

None
