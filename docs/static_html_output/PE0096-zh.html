PE0096: 识别并移除未被使用的局部变量

识别并移除未被使用的局部变量。

缺陷信息

缺陷ID
PE0096
标签
C, CPP ,FIND-BUGS
目录
Checkers/CPP
作者
pinpoint

缺陷描述

未使用的值的存在可能表明这里是一个逻辑错误。为了防止这种错误，应该识别未使用的值并从代码中删除。

漏洞与风险

未使用的值的存在可能表明这里是一个逻辑错误。为了防止这种错误，应该识别未使用的值并从代码中删除。

漏洞利用威胁

低

缓解与预防

识别并移除未被使用的局部变量。

演示实例

漏洞代码示例
int * test(){
  int *p1;
  int *p2;
  p1 = foo();

  return p1;
}

在上面的代码中，局部变量p2从未被使用过。
