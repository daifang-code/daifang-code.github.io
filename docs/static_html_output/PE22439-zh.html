PE22439: W601

.has_key() is deprecated, use 'in'

缺陷信息

缺陷ID
PE22439
标签
flake8
目录
Code style and efficiency
作者
pinpoint

.has_key()在python2中已被弃用。建议使用改为使用in运算符。

示例错误片段
my_dict = {'hello': 'world'}
if my_dict.has_key('hello'):
    print('It works!')
最佳修复
my_dict = {'hello': 'world'}
if 'hello' in my_dict:
    print('It works!')
