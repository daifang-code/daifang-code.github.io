PE1041: switch语句的重复操作

在switch语句中存在多余的位操作。

缺陷信息

缺陷ID
PE1041
标签
CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

在switch语句中存在多余的位操作。

漏洞利用威胁

中

演示实例

漏洞代码示例
switch (x)
     {
         case 2:
          y = a;        // <- this assignment is redundant
         case 3:
          y = b;        // <- case 2 falls through and sets y twice
     }

在上面的代码中，case 2设置了y的值两次。

漏洞代码修复示例
switch (x)
  {
      case 2:
       y = a;break;        
      case 3:
       y = b;break;        
  }

在上面的代码中，case 2将y的值设为2后跳出switch。
