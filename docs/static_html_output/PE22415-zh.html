PE22415: E501

Line too long (82 > 79 characters)

第八条：

限制所需的编辑器窗口宽度可以使多个文件并排打开，并且在使用在相邻列中显示两个版本的代码审阅工具时效果良好。

对于开发人员来说，将最大行长度更改为100或120个字符是相当常见的，尤其是在封闭源代码项目中。
