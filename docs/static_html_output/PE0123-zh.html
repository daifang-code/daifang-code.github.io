PE0123: 可能错误的指针缩放

可能错误的指针缩放。

缺陷信息

缺陷ID
PE0123
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

指针指向的位置不为数组中的元素的起始位置。

漏洞与风险

错误的指针缩放可能导致程序功能错误或缓冲区溢出。

漏洞利用威胁

低

缓解与预防

正确地使用指针缩放。

演示实例

漏洞代码示例
void example_fun()
{
  int a[5] = { 1, 1, 1, 1, 1 };
  char *p = (char *)a;
  //指针"p+3"指向的位置不为整型数组a的元素起始位置
  int b = (int) (*(p+3));  
}
漏洞代码修复示例
void example_fun()
{
  int a[5] = { 1, 1, 1, 1, 1 };
  char *p = (char *)a;
  int b = (int) (*(p+4));  
}
