PE22476: F901

raise NotImplemented should be raise NotImplementedError

缺陷信息

缺陷ID
PE22476
标签
flake8
目录
Code style and efficiency
作者
pinpoint

NotImplemented是一个特殊值，应该由指示操作未实现的二进制特殊方法相对于另一种类型。将NotImplementedError提升到指示未实现超类方法，并且该子级类应该实现它。

示例错误片段
class BaseClass:
    def abstract_method(self):
        raise NotImplemented
最佳修复
class BaseClass:
    def abstract_method(self):
        raise NotImplementedError
