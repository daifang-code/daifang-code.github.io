强制数组方法的回调函数中有 return 语句 (array-callback-return)

Array有一些方法用来过滤、映射和折叠。如果你忘记了在它们的回调函数中写return语句，这种情况可能是个错误。

// example: convert ['a', 'b', 'c'] --> {a: 0, b: 1, c: 2}
var indexMap = myArray.reduce(function(memo, item, index) {
  memo[item] = index;
}, {}); // Error: cannot set property 'b' of undefined

该规则强制数组方法的回调函数中的return语句的使用。

缺陷详因

该规则发现以下方法的回调函数，然后检查return语句的使用。

  • Array.from
  • Array.from
  • Array.prototype.every
  • Array.prototype.every
  • Array.prototype.filter
  • Array.prototype.filter
  • Array.prototype.find
  • Array.prototype.find
  • Array.prototype.findIndex
  • Array.prototype.findIndex
  • Array.prototype.map
  • Array.prototype.map
  • Array.prototype.reduce
  • Array.prototype.reduce
  • Array.prototype.reduceRight
  • Array.prototype.reduceRight
  • Array.prototype.some
  • Array.prototype.some
  • Array.prototype.sort
  • Array.prototype.sort
  • 以上类型的数据。

错误 代码示例：

/*eslint array-callback-return: "error"*/

var indexMap = myArray.reduce(function(memo, item, index) {
    memo[item] = index;
}, {});

var foo = Array.from(nodes, function(node) {
    if (node.tagName === "DIV") {
        return true;
    }
});

var bar = foo.filter(function(x) {
    if (x) {
        return true;
    } else {
        return;
    }
});

正确 代码示例：

/*eslint array-callback-return: "error"*/

var indexMap = myArray.reduce(function(memo, item, index) {
    memo[item] = index;
    return memo;
}, {});

var foo = Array.from(nodes, function(node) {
    if (node.tagName === "DIV") {
        return true;
    }
    return false;
});

var bar = foo.map(node => node.getAttribute("id"));

修改方案

该规则有一个对象选项：

  • "allowImplicit": false (默认) 当设置为 true 时，允许隐式使用 return 不包含任何表达式地返回 undefined。

选项 { "allowImplicit": true } 的 正确 代码示例：

/*eslint array-callback-return: ["error", { allowImplicit: true }]*/
var undefAllTheThings = myArray.map(function(item) {
    return;
});

Known Limitations

该规则检查给定名称的方法中的回调函数，即使该对象中的方法 不是一个数组。

当该情况下该规则可忽略

如果你不想收到关于强制数组方法的回调函数中return语句的使用情况的警告，你可以禁用此规则。

Eslint版本

该规则在 ESLint 2.0.0-alpha-1 中被引入。
