PE22443: W605

Invalid escape sequence 'x'

缺陷信息

缺陷ID
PE22443
标签
flake8
目录
Code style and efficiency
作者
pinpoint

从Python3.6开始，反斜杠字符对不是有效转义序列现在生成一个DeprecationWarning。这个遗嘱最终成为SyntaxError。

示例错误片段
regex = '\.png$'
最佳修复
regex = r'\.png$'
