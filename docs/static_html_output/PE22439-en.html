PE22439: W601

.has_key() is deprecated, use 'in'

Bug info

BugTypeID
PE22439
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

.has_key() was deprecated in Python 2. It is recommended to use the in operator instead.

Anti-pattern
my_dict = {'hello': 'world'}
if my_dict.has_key('hello'):
    print('It works!')
Best practice
my_dict = {'hello': 'world'}
if 'hello' in my_dict:
    print('It works!')
