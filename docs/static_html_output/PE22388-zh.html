PE22388: E222

Multiple spaces after operator

缺陷信息

缺陷ID
PE22388
标签
flake8
目录
Code style and efficiency
作者
pinpoint

操作员之后应该只有一个空间。

示例错误片段
# There are two spaces after the multiplication operator
num = 10
doubled = num *  2
最佳修复
num = 10
doubled = num * 2
