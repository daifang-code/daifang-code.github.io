装箱基础类型（primitive）调用compare方法

漏洞与风险

创建一个装箱基础类型（primitive）只是为了调用compareTo方法。 使用静态比较方法更有效（对于Java 1.4以来的double和float，对于Java 1.7以来的其它基础类型（primitive）），它直接用于基础类型（primitive）。
