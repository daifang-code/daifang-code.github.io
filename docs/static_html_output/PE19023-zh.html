安全风险

    PHP反序列化漏洞(PHP对象注入漏洞)
    传给 `unserialize()`的参数可控 + 构造序列化字符串 --> 从而控制对象内部的变量甚至是函数

修复方案

    严格控制 `unserialize`函数的参数，对`unserialize`后的变量内容进行检查

匹配规则

unserialize
