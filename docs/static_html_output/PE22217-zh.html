DUO107

此linter搜索lxml、xml、xmlrpclib模块的用法。这些库没有针对许多常见的XML攻击进行加固。defusedxml库应该优先于这些模块。

示例错误代码

from xml.etree.ElementTree import parse
et = parse(xmlfile)

正确代码

from defusedxml.ElementTree import parse
et = parse(xmlfile)
