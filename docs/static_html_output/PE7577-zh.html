缺陷描述

MODE_WORLD_READABLE模式文件（Android）

漏洞与风险

在此上下文中所写的文件使用了<code>MODE_WORLD_READABLE</code>的创建模式。 公开所写内容可能程序员的本意。

<b>有风险的代码：</b><br/>

 fos = openFileOutput(filename, MODE_WORLD_READABLE);
 fos.write(userInfo.getBytes());


<b>解决方案(使用MODE_PRIVATE):</b><br/>

 fos = openFileOutput(filename, MODE_PRIVATE);

<b>解决方案(使用本地SQLite数据库):</b><br/>

使用本地SQLite数据库可能是存储结构化数据的最佳解决方案。 确保数据库文件不是在外部存储器上创建的。 请参阅下面的参考资料了解如何实现。
