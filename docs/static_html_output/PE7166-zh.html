缺陷描述

Iterator的next()方法不能抛出NoSuchElementException异常

漏洞与风险

该类实现了java.util.Iterator接口。但是，它的next()方法无法抛出java.util.NoSuchElementException。应该更改next()方法，以便在没有更多元素返回时调用NoSuchElementException。
