禁用与变量同名的标签 (no-label-var)

缺陷详因

该规则旨在通过禁止使用同一作用域下的同名的变量做为标签，来创建更清晰的代码。

错误 代码示例：

/*eslint no-label-var: "error"*/

var x = foo;
function bar() {
x:
  for (;;) {
    break x;
  }
}

正确 代码示例：

/*eslint no-label-var: "error"*/

// The variable that has the same name as the label is not in scope.

function foo() {
  var q = t;
}

function bar() {
q:
  for(;;) {
    break q;
  }
}

当该情况下该规则可忽略

如果你不想收到标签的使用情况的通知，可以关闭此规则。
