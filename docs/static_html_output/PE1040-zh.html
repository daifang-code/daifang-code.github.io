PE1040: 数值处理不当

把指针赋值给整型变量或者把整型变量赋值给指针等不当的数值处理。

缺陷信息

缺陷ID
PE1040
标签
CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

把指针赋值给整型变量或者把整型变量赋值给指针等不当的数值处理。

漏洞与风险

把指针值赋值给整型变量，可能在后续代码由于整形运算，比如乘法，得出无效的指针值。 把整形变量赋值给指针，可能在后续代码引起使用没有声明定义，或者无法预期的内存地址。

漏洞利用威胁

低

缓解与预防

保证整型变量和指针之间不进行互相赋值。

演示实例

漏洞代码示例
int bad(int *p)
{
    int a = p;
    int b = a + 4;
    return b;
}

指针p赋值给整型a。

漏洞代码修复示例
int* foo(int *p)
{
    return p + 4;
}

在上面的代码中，保证指针p不复制给其他整型。
