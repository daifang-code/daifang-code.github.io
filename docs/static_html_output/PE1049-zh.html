PE1049: 错误的异常抛出/捕获

错误的异常抛出/捕获。

缺陷信息

缺陷ID
PE1049
标签
CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

在析够函数里抛异常是危险的，因为析够函数不是程序员主动调用的，很有可能异常没有被catch。

异常值指向释放过的内存。

性能问题，多余的异常信息复制。使用throw exception重新抛出相同异常会复制exception,应该使用throw;替代throw exception免去多余的异常信息的复制。

性能问题，应该使用引用而不是值接收异常信息，这样可以省去不必要的异常信息的复制。

在没有声明会抛出异常的函数抛异常。
