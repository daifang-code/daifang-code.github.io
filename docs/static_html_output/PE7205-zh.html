BE7205: 子类实现了Externalizable接口，但没有定义一个void的构造函数

缺陷描述

子类实现了Externalizable接口，但没有定义一个void的构造函数

漏洞与风险

该类实现了Externalizable接口，但是没有定义一个void构造函数。当Externalizable对象被反序列化时， 它们首先需要通过调用void构造函数。 因为这个类没有一个构造函数，在运行时序列化和反序列化将会失败。
