缺陷描述

在循环中调用NodeList.getLength()

漏洞与风险

该方法在循环中调用NodeList.getlength()，而且NodeList是通过调用getElementsByTagName生成的。这个NodeList并没有存储它的长度，但是每次都不是以最优的方式计算它。考虑在循环之前将长度存储到变量中。

漏洞利用威胁

低
