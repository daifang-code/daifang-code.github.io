PE4022: uninitial ctor

uninitial ctor

Bug info

BugTypeID
PE4022
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

A non-static data member of a class/struct is not initialised by the constructor. It may cause unexpected program behavior if the member is used with an uninitialized value later.

Likelihood of Exploit

Critical

Demonstrative Examples

class Base{
public:
  Base(): m_b(7), m_a(10) { // Compliant

  }

  Base(int a): m_a(a){ // Non-compliant
  }

  int GetmB(void){
    return (m_b);
  }

private:
  int m_a;
  int m_b;
};
