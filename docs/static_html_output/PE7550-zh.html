缺陷描述

Blowfish使用短密钥

漏洞与风险

Blowfish密码支持32位到448位的密钥。 短密钥使密文易受暴力攻击。 如果需要使用Blowfish，则在生成密钥时应使用至少128位熵。

如果可以更改算法，则应使用AES块密码。

易受攻击的代码：

KeyGenerator keyGen = KeyGenerator.getInstance("Blowfish");
keyGen.init(64);

解决方案：

KeyGenerator keyGen = KeyGenerator.getInstance("Blowfish");
keyGen.init(128);
