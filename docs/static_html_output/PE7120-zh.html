缺陷描述

定义协变equals()方法，继承了Object.equals(Object)

漏洞与风险

此类定义了协变的equals()方法，但继承了基本java.lang.Object类中定义的常规equals(Object)方法。该类应该可以定义一个boolean equals(Object)方法。

低
