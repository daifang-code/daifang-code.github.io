DUO124

此linter查找SimpleXMLRPCServer.register\u实例功能。

启用allow_dotted_names选项允许入侵者访问>模块的全局变量，允许入侵者任意执行>你机器上的代码。仅在安全、封闭的网络上使用此选项。

示例错误代码

import SimpleXMLRPCServer

SimpleXMLRPCServer.register_instance(allow_dotted_names=True)

正确代码

import SimpleXMLRPCServer

SimpleXMLRPCServer.register_instance()
