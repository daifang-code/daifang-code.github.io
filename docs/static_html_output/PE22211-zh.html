DUO101

此林楣查找非空的inlineCallbacks'函数returnstatements。使用非空return语句和YOUND`同一函数中的语句是语法错误。

这不是一个安全漏洞，但在构建'twisted'时是一个常见的错误`应用。

示例错误代码

from twisted.internet import defer

@defer.inlineCallbacks
def func(arg):
    return arg

正确代码

from twisted.internet import defer

@defer.inlineCallbacks
def func(arg):
    defer.returnValue(arg)
