缺陷描述

异常被创建并丢弃而不是抛出

漏洞与风险

此代码创建一个异常（或错误）对象，但不对其执行任何操作。例如，

if (x < 0) {
    new IllegalArgumentException("x must be nonnegative");
}

这可能是程序员抛出创建的异常的意图：

if (x < 0) {
    throw new IllegalArgumentException(""x must be nonnegative"");
}
