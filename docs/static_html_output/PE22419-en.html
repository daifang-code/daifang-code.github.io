PE22419: E703

Statement ends with a semicolon

Bug info

BugTypeID
PE22419
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Unlike programming languages like Java and C#, statements in Python do not need to end with a semicolon. They can be removed.

Anti-pattern
print('Hello world!');
Best practice
print('Hello world!')
