BE7121: 定义的equals()方法没有重写Object.equals(Object)

缺陷描述

定义的equals()方法没有重写Object.equals(Object)

漏洞与风险

这个类定义了一个equals()方法不会重写java.lang.Object类中定义的equals(Object)方法。 实际上，这个类应该定义一个boolean equals(Object)方法。
