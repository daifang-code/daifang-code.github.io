禁用 tab (no-tabs)

一些风格指南不允许使用 tab 字符，包括在注释内。

缺陷详因

该规则查找文件中任何位置的 tab：代码，注释或其他。

错误 代码示例：

var a /t= 2;

/**
* /t/t it's a test function
*/
function test(){}

var x = 1; // /t test

正确 代码示例：

var a = 2;

/**
* it's a test function
*/
function test(){}

var x = 1; // test

当该情况下该规则可忽略

如果你已经建立了好的使用 tab 的标准，可以不启用此规则。

Compatibility

  • JSCS: disallowTabs

Eslint版本

该规则在 ESLint 3.2.0 中被引入。
