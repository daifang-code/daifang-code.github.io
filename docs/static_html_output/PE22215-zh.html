DUO105

此linter搜索内置exec函数的用法。这个函数当与用户结合时，通常允许任意代码执行错误输入。

示例错误代码

malicious_user_input = 'open("/etc/passwd", "w").write("bad data")'

# Python 3
exec(malicious_user_input)

# Python 2
exec malicious_user_input

正确代码

def touch_file():
    # Constant argument
    exec('open("/path/to/filename", "w").write("")')

特别提及

如果传递给exec的数据不包含用户输入，则代码可能是安全的如果传递给exec的数据是常量字符串，则代码可能是安全的
