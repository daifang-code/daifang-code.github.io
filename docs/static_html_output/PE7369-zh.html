缺陷描述

在数组上调用equals()，相当于使用==

漏洞与风险

此方法调用数组上的.equals(Object o)方法。由于数组不会重写Object的equals方法，因此在数组上调用equals与比较它们的地址是一样的。要比较数组的内容，请使用java.util.Arrays.equals(Object[], Object[])。要比较数组的地址，使用==显式检查指针相等性会引起更少的混淆。
