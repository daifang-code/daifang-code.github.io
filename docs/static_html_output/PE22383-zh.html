PE22383: E133

Closing bracket is missing indentation

缺陷信息

缺陷ID
PE22383
标签
flake8
目录
Code style and efficiency
作者
pinpoint

右括号缺少缩进。只有在--hang-closingflag，切换的默认行为右括号，以便它们需要悬挂缩进。

示例错误片段

注意：这不是默认的反模式。只有在以下情况下才会发生这种情况使用了--hang-closing标志。

my_list = [
    1, 2, 3,
    4, 5, 6,
]
最佳修复
my_list = [
    1, 2, 3,
    4, 5, 6,
    ]
