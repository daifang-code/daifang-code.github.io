SC1050: Missing 'then' for 'if'

Expected 'then'.

Bug info

BugTypeID
SC1050
Tags
Shell
Category
Checkers/Shell
Authors
pinpoint

Expected 'then'.

Problematic code:
if true
  echo "True"
fi
Correct code:
if true
then
  echo "True"
fi
Rationale:

ShellCheck has found an if statement that appears to be missing a then.

Make sure the then exists, and that it is the first word of the line (or immediately preceded by a semicolon).

Exceptions:

None
