PE4018: identical branches

identical branches

Bug info

BugTypeID
PE4018
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

When the branches of a conditional statement execute exactly the same code, the conditional test is redundant and it may be a copy-paste error by the programmer.

Likelihood of Exploit

Low

Potential Mitigations

Remove identical branches.

Demonstrative Examples

Vulnerable code Example

The following examples are if-else statements with identical branches.

int test_tp1(int x, int y) {
    if (x > 0) {
    	return y;
    }
    return y;
}

int test_tp2(int x, int y) {
    if (x > 0) {
    	return y+2;
    }
    return 2+y;
}


int test_tp3(int x, int y) {
    int z,l;
    if (x > 0) {
    	z = y+2;
    	l = y-1;
    	goto label4;
    }
    z = y+2;
    l = y-1;
label4:
    return z + l;
}
