Backticks are deprecated, use 'repr()'

Bug info

BugTypeID
PE22442
Tags
flake8
Category

Authors

Backticks were deprecated and removed in Python 3. Use the built-in function repr() instead.

Example
obj = MyObj()
print(repr(obj))
