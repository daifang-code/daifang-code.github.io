PE0100: 宏定义参数时括号缺失

宏定义参数时括号缺失。

缺陷信息

缺陷ID
PE0100
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

用宏定义的参数没有使用括号。

漏洞与风险

当某表达式传递给该参数的时，可能产生错误的结果。

漏洞利用威胁

低

缓解与预防

用宏定义参数时应使用括号。

演示实例

漏洞代码示例
#define TRIPLE(a) (a*3)
void example_fun (int a)
{
    // TRIPLE(a + 5)转换为（a + 5*3），与原意不符
    int b = TRIPLE(a + 5); 
    ...
}
漏洞代码修复示例
#define TRIPLE(a) ((a)*3)
void example_fun (int a)
{
    int b = TRIPLE(a + 5); 
    ...
}
