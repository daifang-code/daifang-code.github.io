缺陷描述

定义的equals()方法不会重写equals(Object)

漏洞与风险

此类定义equals()的方法不会重写java.lang.Object中定义的equals(Object)方法。相反，它从超类继承了equals(Object)方法。 该类应该定义一个boolean equals(Object)方法。
