PE0039: 死循环

在循环体中不改变循环变量导致无法退出循环。

缺陷信息

缺陷ID
PE0039
标签
C, CPP ,FIND-BUGS
目录
Checkers/CPP
作者
pinpoint

缺陷描述

在循环体中，如果循环变量不改变，则永远无法退出循环，导致程序死循环。

漏洞与风险

拒绝服务攻击

死循环会导致程序挂起或者崩溃，过多的使用CPU或内存资源，可能会导致拒绝服务攻击。

漏洞利用威胁

低

缓解与预防

在循环体中修改循环变量的值，避免死循环。

演示实例

漏洞代码示例
int global_var = 10;
void bad() {
  while (1) {
    if (global_var==0)
      return;
  }
}

在上面的代码中，循环变量global_var的值在while循环中不会被改变，永远为10，导致死循环。

漏洞代码修复示例
int global_var = 10;
void good() {
  while (1) {
    if (global_var==0)
      return;
    	global_var--;
  }
}

在上面的代码中，在while循环中修改了global_var的值，从而可以使程序退出while循环。
