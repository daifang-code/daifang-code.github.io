Description

Invocation of toString on an unnamed array

Vulnerability and risk

The code invokes toString on an (anonymous) array. Calling toString on an array generates a fairly useless result such as myArray.toString(). Consider using Arrays.toString to convert the array into a readable String that gives the contents of the array.
