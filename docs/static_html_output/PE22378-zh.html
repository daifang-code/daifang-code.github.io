PE22378: E124

Closing bracket does not match visual indentation

缺陷信息

缺陷ID
PE22378
标签
flake8
目录
Code style and efficiency
作者
pinpoint

右括号应与左括号的缩进相匹配。

示例错误片段
result = function_that_takes_arguments('a', 'b', 'c',
                                       'd', 'e', 'f',
)
最佳修复
result = function_that_takes_arguments('a', 'b', 'c',
                                       'd', 'e', 'f',
                                       )
