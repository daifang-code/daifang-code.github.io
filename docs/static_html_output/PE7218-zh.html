BE7218: 不可序列化的类有一个可序列化的内部类

缺陷描述

不可序列化的类有一个可序列化的内部类

漏洞与风险

此Serializable类是不可序列化类的内部类。因此，尝试序列化它的同时也将尝试关联与其相关联的外部类的实例，从而导致运行时错误。

如果可能的话，使内部类成为静态内部类应该可以解决问题。使外部类可序列化也可能有用，但这意味着在序列化内部类的实例时总是会序列化外部类的实例，这通常不符合您的需求。
