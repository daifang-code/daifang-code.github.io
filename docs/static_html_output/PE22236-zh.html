DUO126

此linter搜索popen2模块的使用。

使用popen2模块通常允许执行任意代码与用户输入组合时出现错误。

示例错误代码

import popen2

正确方式

与其使用popen2模块执行命令，不如使用子进程module，同时确保shell=False`。

请参见[用subprocess模块替换旧函数](https://docs.python.org/3/library/subprocess.html\subprocess子流程-替换件）。
