PE0045: 未捕获的异常

一个C++异常被抛出并且没有被处理。

缺陷信息

缺陷ID
PE0045
标签
CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

一个C++异常被抛出并且没有被处理。这样的异常可能会导致程序崩溃或者泄露敏感数据。

漏洞与风险

程序崩溃

一个C++异常被抛出并且没有被处理。这样的异常可能会导致程序崩溃或者泄露敏感数据。

漏洞利用威胁

中

缓解与预防

处理所有的异常情况。

演示实例

漏洞代码示例
void fun() {
  throw 7;
}

int bad(){
  fun();
  return 0;
}

在上面的代码中，在函数bad()中，没有处理fun()抛出的异常。

漏洞代码修复示例
void fun() {
  throw 7;
}

int good(){
  try {
    fun();
  } catch (...) {
    // Handle error
  }
}

在上面的代码中，在函数good()中处理了所有的异常情况。
