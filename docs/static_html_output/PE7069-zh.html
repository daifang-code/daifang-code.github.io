缺陷描述

maps和一组URLs可能会导致性能下降

漏洞与风险

这个方法或字段本身就是，或者使用maps或一组URLs。使用equals方法或者hashCode方法进行资源标识符解析时都会引起堵塞，造成性能问题。 See http://michaelscharf.blogspot.com/2006/11/javaneturlequals-and-hashcode-make.html for more information. 考虑使用java.net.URI 来替代。
