PE22379: E125

Continuation line with same indent as next logical line

缺陷信息

缺陷ID
PE22379
标签
flake8
目录
Code style and efficiency
作者
pinpoint

连续行不应与下一个逻辑行在同一级别缩进。相反，它们应该再缩进一个层次，以便与下一行区分开来。

示例错误片段

在本例中，第二行与下面的线。这让我们很难分辨出if块和布尔表达式的一部分。

if user is not None and user.is_admin or 
    user.name == 'Grant':
    blah = 'yeahnah'
最佳修复
if user is not None and user.is_admin or 
        user.name == 'Grant':
    blah = 'yeahnah'
