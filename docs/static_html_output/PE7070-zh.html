无法使用reflection来检查是否存在没有运行时间保留的注释

缺陷描述

无法使用reflection来检查是否存在没有运行时间保留的注释

漏洞与风险

除非注释本身已使用@Retention(RetentionPolicy.RUNTIME)注释，否则使用reflection无法检查注释 （例如，通过使用isAnnotationPresent方法）。
