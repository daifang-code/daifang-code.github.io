PE1040: Incorrect value operation issue

Assignment address to integer or Assignment integer to address.

Bug info

BugTypeID
PE1040
Tags
CPP
Category
Checkers/CPP
Authors
pinpoint

Description

Assignment address to integer or Assignment integer to address.

Vulnerability and risk

Assigning a pointer to an integer, which may lead to meaningless pointer due to integer operations such as multiplication. Assigning an integer to a pointer, which can lead to dangling pointers that points to unknown region

Likelihood of Exploit

low

Potential Mitigations

Ensures that no assignment between integer and address.

Demonstrative Examples

bad example
int bad(int *p)
{
    int a = p;
    int b = a + 4;
    return b;
}

Assignment address p to integer a.

good example
int* foo(int *p)
{
    return p + 4;
}
