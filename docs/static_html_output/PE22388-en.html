PE22388: E222

Multiple spaces after operator

Bug info

BugTypeID
PE22388
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be only one space after an operator.

Anti-pattern
# There are two spaces after the multiplication operator
num = 10
doubled = num *  2
Best practice
num = 10
doubled = num * 2
