PE22449: F405

Name may be undefined, or defined from star imports: module

缺陷信息

缺陷ID
PE22449
标签
flake8
目录
Code style and efficiency
作者
pinpoint

此警告与问题F403一起出现，'from module import *' used。这意味着变量name可能未定义，但flake8不能确定，因为它也可以进口星空进口。

确认变量是在star导入中定义的。如果是的话，显式地导入它，而不是导入*。如果不是，则定义变量。

示例错误片段
from mymodule import *

def print_name():
    print(name)  # name could be defined in mymodule
最佳修复

显式导入name…

from mymodule import name

def print_name():
    print(name)

…或定义name。

from mymodule import *

def print_name(name):
    print(name)
