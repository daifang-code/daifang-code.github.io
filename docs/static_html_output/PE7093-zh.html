缺陷描述

错误地尝试计算带符号随机整数的绝对值

漏洞与风险

这段代码产生一个随机的有符号整数，然后计算这个随机整数的绝对值。 如果随机数字生成器返回是Integer.MIN_VALUE, 那么结果也会是负的(因为Math.abs(Integer.MIN_VALUE) == Integer.MIN_VALUE). （同样的问题也出现在long类型值上）。
