Unchecked/unconfirmed cast of return value from method

Vulnerability and risk

This code performs an unchecked cast of the return value of a method. The code might be calling the method in such a way that the cast is guaranteed to be safe, but Pinpoint is unable to verify that the cast is safe. Check that your program logic ensures that this cast will not fail.
