PE22412: E306

Expected 1 blank line before a nested definition

Bug info

BugTypeID
PE22412
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Nested functions should contain 1 blank line between their definitions.

Anti-pattern
def outer():
    def inner():
        pass
Best practice
def outer():

    def inner():
        pass
