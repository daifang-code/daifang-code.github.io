缺陷描述

方法调用低效的布尔构造函数，最好改用Boolean.valueOf(...)

漏洞与风险

创建一个新的java.lang.Boolean实例对象会浪费内存，因为Boolean对象是不可变的且只有两个值。最好使用Boolean.valueOf()或者Java1.5中的自动装箱功能来创建一个Boolean实例。
