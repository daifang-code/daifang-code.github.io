强制数组元素间出现换行 (array-element-newline)

(fixable) 命令行中的 --fix 选项可以自动修复一些该规则报告的问题。

很多风格指南要求或禁止在数组元素间出现换行。

缺陷详因

该规则强制数组元素间的换行。

修改方案

该规则有一个字符串选项：

  • "always" (默认) 要求在数组元素间换行
  • "never" 禁止在数组元素间换行

或一个对象选项（只要有任何一个属性满足条件，要求换行。否则，禁止换行）：

  • "multiline": <boolean> 如果数组元素间有换行，则要求换行。如果为 false，该条件不生效。
  • "minItems": <number> 如果数组元素的个数大于等于给定的整数，则要求换行。如果为 0，则该条件将和选项 "always" 一样。如果为 null (默认)，该条件不生效。
always

默认选项 "always" 的 错误 代码示例：

/*eslint array-element-newline: ["error", "always"]*/

var c = [1, 2];
var d = [1, 2, 3];
var e = [
    function foo() {
        dosomething();
    }, function bar() {
        dosomething();
    }
];

默认选项 "always" 的 正确 代码示例：

/*eslint array-element-newline: ["error", "always"]*/

var a = [];
var b = [1];
var c = [1,
    2];
var d = [1,
    2,
    3];
var e = [
    function foo() {
        dosomething();
    },
    function bar() {
        dosomething();
    }
];
never

选项 "never" 的 错误 代码示例：

/*eslint array-element-newline: ["error", "never"]*/

var c = [
    1,
    2
];
var d = [
    1,
    2,
    3
];
var e = [
    function foo() {
        dosomething();
    },
    function bar() {
        dosomething();
    }
];

选项 "never" 的 正确 代码示例：

/*eslint array-element-newline: ["error", "never"]*/

var a = [];
var b = [1];
var c = [1, 2];
var d = [1, 2, 3];
var e = [
    function foo() {
        dosomething();
    }, function bar() {
        dosomething();
    }
];
multiline

选项 { "multiline": true } 的 错误 代码示例：

/*eslint array-element-newline: ["error", { "multiline": true }]*/

var d = [1,
    2, 3];
var e = [
    function foo() {
        dosomething();
    }, function bar() {
        dosomething();
    }
];

选项 { "multiline": true } 的 正确 代码示例：

/*eslint array-element-newline: ["error", { "multiline": true }]*/

var a = [];
var b = [1];
var c = [1, 2];
var d = [1, 2, 3];
var e = [
    function foo() {
        dosomething();
    },
    function bar() {
        dosomething();
    }
];

multiline and minItems

选项 { "multiline": true, "minItems": 3 } 的 错误 代码示例：

/*eslint array-element-newline: ["error", { "multiline": true, "minItems": 3 }]*/

var c = [1,
2];
var d = [1, 2, 3];
var e = [
    function foo() {
        dosomething();
    }, function bar() {
        dosomething();
    }
];

选项 { "multiline": true, "minItems": 3 } 的 正确 代码示例：

/*eslint array-element-newline: ["error", { "multiline": true, "minItems": 3 }]*/

var a = [];
var b = [1];
var c = [1, 2];
var d = [1,
    2,
    3];
var e = [
    function foo() {
        dosomething();
    },
    function bar() {
        dosomething();
    }
];

当该情况下该规则可忽略

如果你不想强制数组元素间的换行，不要启用此规则。
