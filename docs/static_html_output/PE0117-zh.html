PE0117: 初始化成员的顺序不正确

初始化成员的顺序不正确。

缺陷信息

缺陷ID
PE0117
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

在初始化类中的成员时，跟列表中成员顺序不一致。

漏洞与风险

初始化列表成员顺序不正确可能导致部分成员未正确初始化。

漏洞利用威胁

低

缓解与预防

应按列表成员的次序来初始化各成员。

演示实例

漏洞代码示例
class MyClass
{
    MyClass1 mc1;
    MyClass2 mc2;
public:
    //初始化成员的顺序与列表中成员顺序不一致
    MyClass() : MyClass2(1), MyClass1(2) {}  
};
