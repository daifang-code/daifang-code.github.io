缺陷描述

equals()用于比较数组和非数组

漏洞与风险

此方法调用 .equals(Object o)来比较数组和非数组引用。如果被比较的东西是不同类型的，它们将保证不相等，并且比较肯定是错误的。即使它们都是数组，对数组用equals方法也只能确定两个数组是否是同一个对象。要比较数组的内容，请使用java.util.Arrays.equals(Object [], Object [])。
