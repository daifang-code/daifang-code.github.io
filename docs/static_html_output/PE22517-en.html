Message

init method is a generator

Description

Used when the special class method init() is turned into a generator by a yield statement in its body.

This message belongs to the basic checker.

Explanation

The init() method is required to return nothing. Python 2.7 and 3.x raises:

TypeError: init() should return None

when init() is called and executes a yield statement. PyLint reports this error without depending on the actual invocation.
