SC1129: Missing space

You need a space before the !.

Bug info

SC1129
Tags
Shell
Category
Checkers/Shell
Authors
pinpoint

You need a space before the !.

Problematic code:
while! [ -f file ]
do sleep 1; done
Correct code:
while ! [ -f file ]
do sleep 1; done
Rationale:

ShellCheck found a keyword immediately followed by a !. There needs to be a space between them.

Exceptions:

None
