PE0098: 指针类型和非指针类型间进行类型转换

指针类型和非指针类型间进行类型转换。

缺陷信息

缺陷ID
PE0098
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

指针类型和非指针类型间进行类型转换。

漏洞与风险

使用时依赖平台或体系结构，指针可能会也可能不会由相同的比特数作为一个整数类型来表示，如无符号整型，一般情况下，指针类型和非指针类型之间的转换认为是不安全的。

漏洞利用威胁

低

缓解与预防

不宜在一个整型中存储指针值。如果指针指向的类型必须被隐藏，使用void类型代替。

演示实例

漏洞代码示例
void example_fun()
{
  char* a = "hello world";
  unsigned int b = (unsigned int)a;  //指针类型和非指针类型间进行类型转换
  ...
}

在上面的代码中，将指针变量a转换为unsigned int类型。

漏洞代码修复示例
void example_fun()
{
  char* a = "hello world";
  void* b = (void*)a;
  ...
}

在上面的代码中，使用void类型进行指针间类型转换。
