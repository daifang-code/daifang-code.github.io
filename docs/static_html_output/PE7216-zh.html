缺陷描述

可序列化类中的非transient 不可序列化实例字段

漏洞与风险

此Serializable类定义非基本类型实例字段，该字段既不是transient 的，可序列化的，也不是java.lang.Object, 并且似乎不实现Externalizable接口或 readObject() 和writeObject() 方法。  如果在此字段中存储不可序列化对象，则将不能正确反序列化此类的对象。
