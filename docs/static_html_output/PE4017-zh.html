PE4017: 构造函数指针内存泄露

构造函数指针内存泄露

缺陷信息

缺陷ID
PE4017
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

构造函数指针内存泄露

漏洞与风险

构造函数分配了内存，并将指向它的指针存储在对象字段中，但析构函数没有释放该内存。

漏洞利用威胁

严重

缓解与预防

添加释放该字段的析构函数语句。

演示实例

Vulnerable code Example
class A{
  int *p;

  A() {
    p = new char[1024];
  }
  ~A()
  {
    // p指向的內存可能泄漏
  }
}
