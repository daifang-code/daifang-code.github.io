PE0096: Detect and remove unused values

Detect and remove unused values.

Bug info

BugTypeID
PE0096
Tags
C, CPP ,FIND-BUGS
Category
Checkers/CPP
Authors
pinpoint

Description

The presence of unused values may indicate significant logic errors. To prevent such errors, unused values should be identified and removed from code.

Vulnerability and risk

The presence of unused values may indicate significant logic errors. To prevent such errors, unused values should be identified and removed from code.

Likelihood of Exploit

Low

Potential Mitigations

Detect and remove unused values.

Demonstrative Examples

Vulnerable Code Example
int * test(){
  int *p1;
  int *p2;
  p1 = foo();

  return p1;
}

In the vulnerable code example, local variable p2 is not used.
