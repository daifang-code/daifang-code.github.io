PE4013: Hidden password echo

The password field is not masked when the user enters a password.

Bug info

BugTypeID
PE4013
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

The password field is not masked when the user enters a password.

Vulnerability and risk

Increased the likelihood that an attacker will get a password by observing the screen.

Likelihood of Exploit

Medium

Potential Mitigations

The password field is masked when the user enters a password. Usually, every character entered by the user should be echoed as an asterisk.

Demonstrative Examples

Vulnerable Code Example
void example_fun()
{
    char password[MAX_STR_LEN];  //The value of MAX_STR_LEN is 15
    int i = 0;
    char c = '0';
    printf("Please enter the password: ");
    while(i < MAX_STR_LEN && c!='\n')
    {
        scanf("%c",&c);  //Did not mask the password entered by the user
        password[i] = c;
        i++;
    }
}
Fixed code example
void example_fun()
{
    char password[MAX_STR_LEN];  //The value of MAX_STR_LEN is 15
    int i = 0;
    char c = '0';
    printf("Please enter the password: ");
    while(i < MAX_STR_LEN && (c=getch())!='\n')
    {
        password[i] = c;
        putchar('*');  //Replace the password entered by the user with an asterisk
        i++;
    }
}
