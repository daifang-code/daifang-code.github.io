缺陷描述

弱SSLContext

漏洞与风险

易受攻击的代码：

    SSLContext.getInstance("SSL");

解决方案：

将安装启用升级到以下内容，并配置https.protocols JVM选项以包含TLSv 1.2：</p>

    SSLContext.getInstance("TLS");
