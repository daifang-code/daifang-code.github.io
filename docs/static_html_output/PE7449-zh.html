缺陷描述

没有使用类型限定符的值在需要具有该限定符的值时使用

漏洞与风险

一个值被使用的方式需要用类型限定符来注释。类型限定符是严格的，因此该工具拒绝任何没有适当注释的值。

为了强制一个值有一个严格的注释，定义一个恒等函数，其中返回值用严格的注释标注。这是将一个不带注释的值转换为一个严格类型限定符注释值的唯一方法。
