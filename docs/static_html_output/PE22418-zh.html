PE22418: E702

Multiple statements on one line (semicolon)

缺陷信息

缺陷ID
PE22418
标签
flake8
目录
Code style and efficiency
作者
pinpoint

多个语句不应在同一行上用分号分隔。他们应该各自为政

示例错误片段
from gevent import monkey; monkey.patch_all()
最佳修复
from gevent import monkey
monkey.patch_all()
