缺陷描述

在对象的输出流处添加元素

漏洞与风险

这段代码在append模式下打开一个文件，然后将结果包装在一个对象输出流中。 这将不允许您附加存储在文件中的现有对象输出流。 如果您希望能够附加到一个对象输出流中，您需要保持对象输出流的打开。

在append模式下打开文件和编写对象输出流的唯一情况是在读取一个文件时，您计划以随机访问模式打开它，并在append启动时寻求字节偏移。
