PE22409: E303

Too many blank lines (3)

缺陷信息

缺陷ID
PE22409
标签
flake8
目录
Code style and efficiency
作者
pinpoint

两个类之间应该有一个空行，类之间应该有一个空行。

示例错误片段

这个例子有太多的空行。

def func1():
    pass



def func2():
    pass
最佳修复
def func1():
    pass


def func2():
    pass
