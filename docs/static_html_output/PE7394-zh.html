缺陷描述

int除法结果为double或float

漏洞与风险

此代码将整数除法（例如，int或long除法）运算的结果转换为double或float。 对整数进行除法会将结果截断为最接近零的整数值。将结果转换为double表明应保留此精度。 可能意味着在执行除法之前应将一个或两个操作数强制转换为double。以下是一个例子：

int x = 2;
int y = 5;
// 错误：输出结果为0.0
double value1 = x / y;
// 正确：输出结果为0.4
double value2 = x / (double) y;
