缺陷描述

HTTP头不可信

漏洞与风险

请求头可以很容易地由请求用户修改。一般来说，不应该假设请求来自于没有被攻击者修改过的常规浏览器。因此，建议您不要在任何关于请求的安全决策中相信这个值。
