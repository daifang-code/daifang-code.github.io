Description

Invocation of toString on an array

Vulnerability and risk

The code invokes toString on an array, which will generate a fairly useless result such as <code>myArray.toString()</code>. Consider using <code>Arrays.toString</code> to convert the array into a readable String that gives the contents of the array.
