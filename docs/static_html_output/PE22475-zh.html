PE22475: F841

Local variable name is assigned to but never used

缺陷信息

缺陷ID
PE22475
标签
flake8
目录
Code style and efficiency
作者
pinpoint

函数中的局部变量已定义，但未被使用。应删除此局部变量。

示例错误片段
def full_name(self):
    # name is defined but not used
    name = self.first_name + ' ' + self.last_name
    return self.first_name
