PE22394: E241

Multiple spaces after ','

Bug info

BugTypeID
PE22394
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be only one space after the , character.

Anti-pattern

In this example there are two spaces after the comma

my_tuple = 1,  2
Best practice
my_tuple = 1, 2
