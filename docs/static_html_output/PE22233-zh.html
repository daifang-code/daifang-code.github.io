DUO123

这个linter查找Pythonrequests库的不安全使用。

这个库已经成为发出HTTP请求的事实标准。这个linter专门搜索使用HTTPS请求的情况verifycertificateverifyvia the keywordverification=False关闭。

示例错误代码

import requests

request = requests.get("https://google.com", verify=False)

正确代码

import requests

request = requests.get("https://google.com")
