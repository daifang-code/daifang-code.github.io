PE22391: E225

Missing whitespace around operator

Bug info

BugTypeID
PE22391
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be one space before and after all operators.

Anti-pattern
if age>15:
    print('Can drive')
Best practice
if age > 15:
    print('Can drive')
