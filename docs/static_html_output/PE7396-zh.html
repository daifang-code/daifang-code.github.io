缺陷描述

int值强制转换为float，然后传递给Math.round

漏洞与风险

此代码将int值转换为float精度浮点数，然后将结果传递给Math.round()函数，该函数返回最接近参数的int/long。此操作应始终为无操作，因为将整数转换为float应该给出一个没有小数部分的数字。 生成要传递给Math.round的值的操作可能是使用浮点计算执行的。
