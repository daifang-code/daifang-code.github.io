PE22414: E402

Module level import not at top of file

缺陷信息

缺陷ID
PE22414
标签
flake8
目录
Code style and efficiency
作者
pinpoint

所有模块级导入都应位于文件的顶部。这意味着模块级导入之间不应该有语句。

示例错误片段

在本例中，sysimport不在文件的顶部因为local.setlocale出现在它之前。

import locale

locale.setlocale(locale.LC_ALL, 'en_US.UTF-8')

import sys
最佳修复

更改代码，使方法调用在导入之后发生。

import locale
import sys

locale.setlocale(locale.LC_ALL, 'en_US.UTF-8')
