PE22384: E201

Whitespace after '('

缺陷信息

缺陷ID
PE22384
标签
flake8
目录
Code style and efficiency
作者
pinpoint

左括号前后不能有空格。

示例错误片段
# The space after open is unnecessary
with open( 'file.dat') as f:
    contents = f.read()
最佳修复
with open('file.dat') as f:
    contents = f.read()
