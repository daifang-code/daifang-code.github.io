缺陷描述

没有填充的RSA是不安全的

漏洞与风险

该软件使用RSA算法，但未包含最佳非对称加密填充（OAEP），这可能会削弱加密。

缺陷代码：

Cipher.getInstance("RSA/NONE/NoPadding")

解决方案：
代码应替换为：

Cipher.getInstance("RSA/ECB/OAEPWithMD5AndMGF1Padding")
