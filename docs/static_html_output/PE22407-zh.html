PE22407: E301

Expected 1 blank line, found 0

缺陷信息

缺陷ID
PE22407
标签
flake8
目录
Code style and efficiency
作者
pinpoint

类的方法之间应有一个空行。

示例错误片段
class MyClass(object):
    def func1():
        pass
    def func2():
        pass
最佳修复
class MyClass(object):
    def func1():
        pass

    def func2():
        pass
