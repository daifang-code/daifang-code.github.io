PE22385: E202

Whitespace before ')'

Bug info

BugTypeID
PE22385
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Closing parentheses should not have any before after them.

Anti-pattern
with open('file.dat' ) as f:
    contents = f.read()
Best practice
with open('file.dat') as f:
    contents = f.read()
