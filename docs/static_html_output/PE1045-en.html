Description

A private function is never used. Is it really useful, or should be 'public'?

Likelihood of Exploit

low

Demonstrative Examples

bad example
class Foo
{
  public:
    Foo& operator++();
  private:
    void Bar() const {
    }
};

Foo& Foo::operator++()
{
  Bar();
  return *this;
}

In this code，Foo::Bar is unused private fuction。
