缺陷描述

潜在的外部配置控制

漏洞与风险

允许外部控制系统设置可能会中断服务或导致应用程序以恶意方式运行。 攻击者可能通过提供不存在的目录名称或连接到数据库的未授权部分，导致程序出错。


风险代码：

conn.setCatalog(request.getParameter("catalog"));

高
