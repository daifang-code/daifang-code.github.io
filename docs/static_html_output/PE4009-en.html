PE4009: Reversible hash function

The hash algorithm used by the software has a reversible algorithm that can use the generated hash value to determine the original input, or can find an input that produces the same hash value.

Bug info

BugTypeID
PE4009
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

The hash algorithm used by the software has a reversible algorithm that can use the generated hash value to determine the original input, or can find an input that produces the same hash value.

Vulnerability and risk

An attacker can obtain the original input or the input that produces the same hash value through a reverse algorithm, thereby bypassing the security authentication of the hash-dependent algorithm.

Likelihood of Exploit

Medium

Potential Mitigations

Employe the currently accepted irreversible standard hashing algorithm.
