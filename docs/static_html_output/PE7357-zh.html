缺陷描述

TestCase声明了一个糟糕的suite方法

漏洞与风险

Class是一个JUnit TestCase并定义了suite()方法。然而，这个suite方法也需要被声明为

public static junit.framework.Test suite()

或

public static junit.framework.TestSuite suite()

漏洞利用威胁

低
