PE0130: Unused lable

The label was not used after it was defined.

Bug info

BugTypeID
PE0130
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

The label was not used after it was defined.

Vulnerability and risk

Useless tags have no meaning and may result in incorrect program functionality.

Likelihood of Exploit

Low

Potential Mitigations

Useless tags should be removed.

Demonstrative Examples

Vulnerable Code Example
int example_fun(int a)
{
  if(a > 0)
  {
    a++;
    return a;
  }
  else
  {
    la:  //Label la is not used
      printf("Error!")
      return 0;
  }
}
Fixed Code Example
int example_fun(int a)
{
  if(a > 0)
  {
    a++;
    return a;
  }
  else
  {
    printf("Error!")
    return 0;
  }
}
