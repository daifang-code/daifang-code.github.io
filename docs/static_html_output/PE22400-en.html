PE22400: E265

Block comment should start with '# '

Bug info

BugTypeID
PE22400
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Block comments should have one space before the pound sign (#) and the comment itself.

Anti-pattern
#This comment needs a space
def print_name(self):
    print(self.name)
Best practice
# Comment is correct now
def print_name(self):
    print(self.name)
