PE22411: E305

Expected 2 blank lines after end of function or class

缺陷信息

缺陷ID
PE22411
标签
flake8
目录
Code style and efficiency
作者
pinpoint

函数和类后面应该有两个空行，将它们与其他函数和类分开。

示例错误片段
class User(object):
    pass
user = User()
最佳修复
class User(object):
    pass


user = User()
