PE0102: 有符号的一位位域

位域带符号且仅为一位。

缺陷信息

缺陷ID
PE0102
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

位域带符号且仅为一位。

漏洞与风险

有符号的一位位域不具有数值意义，可能导致程序功能实现不正确。

漏洞利用威胁

低

缓解与预防

有符号的位域应至少有两位。

演示实例

漏洞代码示例
struct S
{
  int x:1;
};

void example_fun(struct S a) 
{
  a.x = 1;
  if (a.x == 1)  //  条件为假，因为'a.x' 为-1 
  {
    //...
  }
}
