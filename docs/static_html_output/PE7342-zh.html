缺陷描述

在util.concurrent抽象中使用监视器样式wait方法

漏洞与风险

这个方法在一个对象上调用wait(),notify()或notifyAll()，也提供了await(),signal(),signalAll()方法（例如util.concurrent 条件对象)。这可能不是您想要的，如果这真的是您想要的，请您考虑改变您的编程方式，因为其他开发人员会认为这一编程方法非常令人困惑。
