PE22374: E111

Indentation is not a multiple of four

Bug info

BugTypeID
PE22374
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

PEP8 recommends that Python code indentation be a multiple of four.

Anti-pattern
class User(object):
  def __init__(self, name):
    self.name = name
Best practice
class User(object):
    def __init__(self, name):
        self.name = name
