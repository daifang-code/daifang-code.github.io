PE22373: E101

Indentation contains mixed spaces and tabs

缺陷信息

缺陷ID
PE22373
标签
flake8
目录
Code style and efficiency
作者
pinpoint

缩进中有制表符和空格。程序员应该使用制表符或空格，但不是两者 .

示例错误片段

*注意：•字符代表一个空格，→字符表示制表符。

在本例中，第三行包含四个空格和两个制表符。

def get_name(self):
    if self.first_name and self.last_name:
••••→ → return self.first_name + ' ' + self.last_name
    else:
        return self.last_name
最佳修复

将该行改为仅使用空格。

def get_name(self):
    if self.first_name and self.last_name:
••••••••return self.first_name + ' ' + self.last_name
    else:
        return self.last_name
