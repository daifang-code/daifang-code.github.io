PE22403: E272

Multiple spaces before keyword

Bug info

BugTypeID
PE22403
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be only one space before a keyword.

Anti-pattern

In this example there are two spaces before the in keyword.

def func():
    if 1  in [1, 2, 3]:
        print('yep!')
Best practice
def func():
    if 1 in [1, 2, 3]:
        print('yep!')
