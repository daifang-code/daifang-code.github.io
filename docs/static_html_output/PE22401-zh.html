PE22401: E266

Too many leading '#' for block comment

缺陷信息

缺陷ID
PE22401
标签
flake8
目录
Code style and efficiency
作者
pinpoint

块注释应该只有一个前导的#。

示例错误片段
## Prints hello
print('hello')
最佳修复

只使用一个#。

# Prints hello
print('hello')
