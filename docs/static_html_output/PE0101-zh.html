PE0101: 内置类型和布尔值之间的隐式转换

内置类型和布尔值之间的隐式转换。

缺陷信息

缺陷ID
PE0101
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

此检查可用于查找内置类型和布尔值之间的隐式转换。根据使用情况，它可能仅有助于提高代码的可读性，或者在某些情况下，指出由于隐式转换而未引起注意的潜在错误。

漏洞与风险

boolean类型的变量可以并且只能取两个值中的一个，即true或false，对boolean类型的变量进行隐式类型转换或者算术运算无意义，可能导致程序功能实现不正确。

漏洞利用威胁

低

缓解与预防

不对boolean类型与内置类型进行相互隐式转换。

演示实例

漏洞代码示例
void example_fun(bool a)
{
  a = a + 2;  // boolean类型变量的加法运算
}
