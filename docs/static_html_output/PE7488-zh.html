缺陷描述

Cookie中潜在的敏感数据

漏洞与风险

存储在自定义cookie中的信息不应该是敏感的或与会话相关的。 在大多数情况下，敏感数据应该只存储在会话中，并由用户的会话cookie引用。 参考HttpSession(HttpServletRequest.getSession()) 定制cookie可以用于需要比特定会话独立的信息。

定制cookie可以被用于需要比特定会话存在时间长的并且独立于一个特定会话的信息中。

漏洞利用威胁

高
