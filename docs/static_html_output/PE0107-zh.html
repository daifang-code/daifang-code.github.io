PE0107: 在用于控制程序流程的条件表达式中使用位运算符'&'或'|'

在用于控制程序流程的条件表达式中使用位运算符'&'或'|'。

pinpoint

缺陷描述

在用于控制程序流程的条件表达式中使用位运算符'&'或'|'。

漏洞与风险

当用于控制程序流程的条件表达式使用位运算符'&'或'|'时，程序的本意可能为使用运算符"&&"或"||"，可能会改变原本的判断逻辑，导致程序执行不同的分支路径。

漏洞利用威胁

低

缓解与预防

使用运算符'&&'或'||'。

演示实例

漏洞代码示例
void example_fun(int a) 
{
  //在用于控制程序流程的条件表达式中使用位运算符“&”
  if (a >0  & a < 20) 
  {  
    ...
  }
}
漏洞修复代码示例
void example_fun(int a) 
{
  if (a >0  && a < 20) 
  {  
    ...
  }
}
