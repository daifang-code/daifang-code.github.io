PE22375: E117

Over-indented

缺陷信息

缺陷ID
PE22375
标签
flake8
目录
Code style and efficiency
作者
pinpoint

代码应该有一致的缩进，通常以两到四个增量隔开。

示例错误片段

下一行的print函数缩进为5空格而不是四个。

if True:
     print('Hi there')
最佳修复

代码现在的间隔是四个空格而不是五个空格。

if True:
    print('Hi there')
