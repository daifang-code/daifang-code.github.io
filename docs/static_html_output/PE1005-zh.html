PE1005: 将非结构体类型转换为结构体类型

将非结构类型转换为结构类型并访问内部字段会导致内存访问错误或数据损坏。

缺陷信息

缺陷ID
PE1005
标签
C, C++, CWE-588
目录
Checkers/CPP
作者
pinpoint

缺陷描述

将非结构类型转换为结构类型并访问内部字段会导致内存访问错误或数据损坏。

漏洞与风险

由于内存访问错误，执行可能会结束。

缓解与预防

禁止不兼容的类型转换。

漏洞利用威胁

中

演示示例

漏洞缺陷示例
struct foo
{
  int i;
};

int main(int argc, char **argv)
{
  *foo = (struct foo *)main;
  foo->i = 2;
  return foo->i;
}
