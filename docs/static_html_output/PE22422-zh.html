PE22422: E714

Test for object identity should be 'is not'

缺陷信息

缺陷ID
PE22422
标签
flake8
目录
Code style and efficiency
作者
pinpoint

对象标识测试应使用x is not None而不是not x is None。前一个例子更简单可读。

示例错误片段
if not user is None:
    print(user.name)
最佳修复
if user is not None:
    print(user.name)
