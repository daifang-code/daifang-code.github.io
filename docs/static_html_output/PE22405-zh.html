PE22405: E274

Tab before keyword

缺陷信息

缺陷ID
PE22405
标签
flake8
目录
Code style and efficiency
作者
pinpoint

关键字前应该只有一个空格。

示例错误片段

在本例中，in关键字前有一个选项卡。

*注意：→字符表示制表符。

def func():
    if 1→in [1, 2, 3]:
        print('yep!')
最佳修复

在in运算符前只使用一个空格。

def func():
    if 1 in [1, 2, 3]:
        print('yep!')
