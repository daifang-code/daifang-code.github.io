缺陷描述

巨大的字符串常量跨多个类文件复制

漏洞与风险

一个大的字符串常量被复制到多个类文件中。

这很可能是因为最后一个字段被初始化为字符串常量，Java语言强制将对其它类的最后一个字段的所有引用都内联到此类文件中。 见 <a href="http://bugs.java.com/bugdatabase/view_bug.do?bug_id=6447475">JDK bug 6447475</a> 关于JDK中此错误发生的描述以及如何解决这个问题。
