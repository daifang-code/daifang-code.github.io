PE22396: E251

Unexpected spaces around keyword / parameter equals

缺陷信息

缺陷ID
PE22396
标签
flake8
目录
Code style and efficiency
作者
pinpoint

函数中的=前后不应有空格定义。

示例错误片段
def func(key1 = 'val1',
         key2 = 'val2'):
    return key1, key2
最佳修复
def func(key1='val1',
         key2='val2'):
    return key1, key2
