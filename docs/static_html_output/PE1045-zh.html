PE1045: 未使用的私有函数

未使用的私有函数，请确定它是否虚需要存在，或者应该设为public?

缺陷信息

缺陷ID
PE1045
标签
CPP ,FIND-BUGS
目录
Checkers/CPP
作者
pinpoint

缺陷描述

未使用的私有函数，请确定它是否虚需要存在，或者应该设为public?这种缺陷导致程序执行效率低下。

漏洞利用威胁

低

演示实例

漏洞代码示例
class Foo
{
  public:
    Foo& operator++();
  private:
    void Bar() const {
    }
};

Foo& Foo::operator++()
{
  Bar();
  return *this;
}

在上面的代码中，Foo::Bar为未使用的私有函数。
