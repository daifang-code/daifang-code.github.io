PE0120: switch语句缺少default分支

switch语句缺少default分支。

缺陷信息

缺陷ID
PE0120
标签
C, CPP ,FIND-BUGS
目录
Checkers/CPP
作者
pinpoint

缺陷描述

在switch代码块中，缺少default逻辑分支。

漏洞与风险

代码的可读性可维护性降低，可能导致逻辑上的缺失。

漏洞利用威胁

低

缓解与预防

应确保每个switch语句块都有default分支。

演示实例

漏洞代码示例
void example_fun(int a)
{
  switch (a) 
  {
    case 0:
      ... 
      break;
    case 1:
      ...
      break;
  }  //switch语句缺少default分支
}
漏洞代码修复示例
void example_fun(int a)
{
  switch (a) 
  {
    case 0:
      ... 
      break;
    case 1:
      ...
      break;
    default:
      ...
  }  
}
