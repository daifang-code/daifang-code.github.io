PE22434: W293

Blank line contains whitespace

缺陷信息

缺陷ID
PE22434
标签
flake8
目录
Code style and efficiency
作者
pinpoint

空行不应包含任何制表符或空格。

示例错误片段

*注意：•字符代表一个空格。

def first_func():
    pass
••••  # This line contains four spaces

def second_func():
    pass
最佳修复
def first_func():
    pass


def second_func():
    pass
