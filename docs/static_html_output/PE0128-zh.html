PE0128: 参数被局部变量隐藏

参数被局部变量隐藏是指参数和局部变量同名的现象。

缺陷信息

缺陷ID
PE0128
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

参数被局部变量隐藏是指参数和局部变量同名的现象。

漏洞与风险

局部变量与函数参数同名时，会使这些函数参数失效，这可能并不是程序员的本意，可能导致程序功能实现不正确。

漏洞利用威胁

低

缓解与预防

在编写程序时，不应出现局部变量与参数同名。

演示实例

漏洞代码示例
void example_fun(int a) 
{
    int a = 1;     // 局部变量与函数参数同名
    ...
}
漏洞代码修复示例
void example_fun(int a) 
{
    int b = 1; 
    ...
}
