安全风险

    LDAP Injection
    允许进行LDAP查询 + 输入未进行过滤 ---> LDAP注入
    这种威胁可以让攻击者能够从LADP树中提取到很多很重要的信息

修复方案

    对用户输入数据中包含的”语言本身的保留字符”进行转义(例如可以使用`ldap_escape`)

匹配规则

(ldap_add|ldap_delete|ldap_list|ldap_read|ldap_search|ldap_bind)
