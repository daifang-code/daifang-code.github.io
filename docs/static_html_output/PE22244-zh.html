DUO134

此linter搜索cryptography模块的不安全属性使用。

Pythoncryptography库已经成为晶体手术。众所周知，加密操作非常困难为了得到正确的答案，经常会遇到很多麻烦。这个林特在寻找cryptography库中不安全的加密原语和操作。

示例错误代码

cryptography.hazmat.primitives.hashes.MD5
cryptography.hazmat.primitives.hashes.SHA1
cryptography.hazmat.primitives.ciphers.modes.ECB
cryptography.hazmat.primitives.ciphers.algorithms.Blowfish
cryptography.hazmat.primitives.ciphers.algorithms.ARC4
cryptography.hazmat.primitives.ciphers.algorithms.IDEA

Dlint also looks for PKCS1 v1.5 usage via the following attribute:

cryptography.hazmat.primitives.asymmetric.padding.PKCS1v15

正确代码

对于签名操作：密码学.危险品.原语.不对称.padding.PSS对于加密操作：密码学.危险品.原语.非对称.padding.OAEP

特别提及

*PKCS1v15可用于旧版应用程序，但不应考虑对于新的应用程序。仍然建议您远离PKCS1v15尽快使用。
