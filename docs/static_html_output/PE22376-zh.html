PE22376: E121

Continuation line under-indented for hanging indent

缺陷信息

缺陷ID
PE22376
标签
flake8
目录
Code style and efficiency
作者
pinpoint

行的缩进量比悬挂缩进时少。

示例错误片段

在第一个示例中，dict的键只缩进有3个空格。它们应该用4个空格缩进。

result = {
   'key1': 'value',
   'key2': 'value',
}
最佳修复
result = {
    'key1': 'value',
    'key2': 'value',
}
示例错误片段

在本例中，第二行仅缩进2个空格。它应该用4个空格缩进。

print("Python", (
  "Rules"))
最佳修复
print("Python", (
    "Rules"))
