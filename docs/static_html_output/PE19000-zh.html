安全风险

    不安全的权限设置
    权限为0777表示所有权限均开放，即属性为:drwxrwxrwx(用户可读可写可执行)。

修复方案

    设置相应的权限，尽量减少权限为0777。

匹配规则

(mkdir|chmod)\s(\s(.?)\s,\s0777\s)
