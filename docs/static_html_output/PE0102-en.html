Description

One bit field with sign

Vulnerability and risk

A signed one bit field is not numerical and may result in incorrect program functionality.

Likelihood of Exploit

Low

Potential Mitigations

A signed bit field should have at least two bits.

Demonstrative Examples

Vulnerable Code Example
struct S
{
  int x:1;
};

void example_fun(struct S a) 
{
  a.x = 1;
  if (a.x == 1)  //  The condition is false because 'a.x' is -1
  {
    //...
  }
}
