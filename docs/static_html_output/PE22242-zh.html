DUO132

此linter搜索urllib3模块中使用的不安全关键字参数。具体来说，它查找可能具有HTTPS认证的对象通过设置无、证书无、证书无或ssl.CERT_无为cert_reqskeyward参数设置。这些值禁用验证，从而允许不安全的HTTPS连接。

示例错误代码

import urllib3
import ssl
from ssl import CERT_NONE

urllib3.PoolManager(cert_reqs="CERT_NONE")
urllib3.ProxyManager(cert_reqs="CERT_NONE")
urllib3.HTTPSConnectionPool(cert_reqs="NONE")
urllib3.connection_from_url(cert_reqs=ssl.CERT_NONE)
urllib3.proxy_from_url(cert_reqs=CERT_NONE)

正确方式

简单地不指定cert_reqs将默认为安全行为。此外，设置需要ssl.CERT_或者它的变种将确保验证执行。
