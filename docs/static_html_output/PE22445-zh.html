PE22445: F401

Module imported but unused

缺陷信息

缺陷ID
PE22445
标签
flake8
目录
Code style and efficiency
作者
pinpoint

模块已导入，但未在文件中的任何位置使用。应该使用该模块或删除导入。

示例错误片段

在本例中，namedtuple。但是，tuple是从未使用过。它应该被移除。

from collections import namedtuple

my_tuple = ('Grant', 'McConnaughey', 25)
最佳修复
my_tuple = ('Grant', 'McConnaughey', 25)
