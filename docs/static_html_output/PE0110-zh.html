PE0110: 条件判别成立时相应分支无执行语句

条件判别成立时相应分支无执行语句。

缺陷信息

缺陷ID
PE0110
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

条件判别成立时相应分支无任何执行语句。

漏洞与风险

这通常不是程序的本意，可能是由于疏忽而遗漏掉了语句，可能会导致程序出错或程序功能实现不正确。

漏洞利用威胁

低

缓解与预防

条件判别成立时相应分支不能为空。

演示实例

漏洞代码示例
void example_fun(int a) 
{
  if(a > 0)
  {
    //条件判别成立时相应分支无执行语句
  }
}
漏洞修复代码示例
void example_fun(int a) 
{
  if(a > 0)
  {
    a++;
  }
}
