缺陷描述

“.”或“|”用于正则表达式

漏洞与风险

正在调用String函数，并将"."或"|"传递给以正则表达式作为参数的参数。这是程序员的真实意图吗？ 例如

  • s.replaceAll(".", "/")将返回一个字符串，其中每个字符都被'/'字符替换
  • s.split(".") 总是返回一个零长度的String数组
  • "ab|cd".replaceAll("|", "/")将返回"/a/b/|/c/d/"
  • "ab|cd".split("|")将返回带有六个元素的数组： [, a, b, |, c, d]
