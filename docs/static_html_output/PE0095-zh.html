PE0095: 释放空指针

释放空指针是一个无效的操作。

缺陷信息

缺陷ID
PE0095
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

void free (void* ptr); // Deallocate memory block

调用free函数用来释放之前通过调用malloc，calloc或realloc分配的内存块，使其可以再次用于进一步的分配。
如果ptr没有指向使用上述函数分配的内存块，则会导致未定义的行为。
如果ptr是一个空指针，该函数什么也不做。

漏洞与风险

释放空指针是一个无效的操作。

漏洞利用威胁

低

缓解与预防

不应该释放一个空指针。

演示实例

漏洞代码示例
void test(){
  free(0);
}

在上面的代码中，通过free()函数释放了nullptr。
