PE0073: 自我赋值

变量赋值给自己。

缺陷信息

缺陷ID
PE0073
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

自我赋值通常是一个打字错误，可能会导致程序出错或程序功能实现不正确。

漏洞与风险

意料之外的结果

自我赋值通常是一个打字错误，可能会导致程序出错或程序功能实现不正确。

漏洞利用威胁

低

缓解与预防

检查代码逻辑，删除无效的自赋值代码。

演示实例

漏洞代码示例
void bad() {
  int a, b;
  a = 10;
  a = a;
}

在上面的代码中，第4行是无效的自赋值代码。

漏洞代码修复示例
void good() {
  int a, b;
  a = 10;
  b = a;
}

在上面的代码中，移除无效的自赋值代码。
