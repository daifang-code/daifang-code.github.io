PE0097: 可疑的数组到指针的转换

可疑的数组到指针的转换。

缺陷信息

缺陷ID
PE0097
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

使用数组名去访问数组中的元素，数组名转化为指针，通常由缺少索引数据引起错误。

漏洞与风险

访问不希望的数据，可能导致执行结果错误。

漏洞利用威胁

低

缓解与预防

应使用索引号来访问数组元素。

演示实例

漏洞代码示例
struct S
{
  char c; 
};
void example_fun()
{
  struct S a[5];
  for (int i = 0; i < 5; i++)
  {
    a->c = 'a'; //数组名转化为指针，始终访问的是第一个元素
  }
}

在上面的代码中，数组名转化为指针，始终访问的是第一个元素。

漏洞代码修复示例
struct S
{
  char c; 
};
void example_fun()
{
  struct S a[5];
  for (int i = 0; i < 5; i++)
  {
    a[i]->c = 'a'; 
  }
}

在上面的代码中，使用索引号来访问数组元素。
