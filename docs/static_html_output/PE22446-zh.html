PE22446: F402

Import module from line n shadowed by loop variable

缺陷信息

缺陷ID
PE22446
标签
flake8
目录
Code style and efficiency
作者
pinpoint

循环变量不应重新定义导入的模块的名称。

示例错误片段

在本例中，path从path导入的模块。循环变量的名称应该改变。

from os import path

for path in ['file1.py', 'file2.py']:
    print(path)
最佳修复
from os import path

for file_name in ['file1.py', 'file2.py']:
    print(file_name)
