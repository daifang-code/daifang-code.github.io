DUO125

此linter搜索commands模块的使用情况。

使用commands模块通常允许执行任意代码与用户输入组合时出现错误。

示例错误代码

import commands

commands.getstatusoutput('ls /bin/ls')

正确方式

与其使用commands模块来执行命令，不如使用子进程module，同时确保shell=False`。

请参见[用subprocess模块替换旧函数](https://docs.python.org/3/library/subprocess.html\subprocess子流程-替换件）。
