SC1069: Missing space

You need a space before the [.

Bug info

Authors

You need a space before the [.

Problematic code:
if[ -e file ]
then echo "exists"; fi
Correct code:
if [ -e file ]
then echo "exists"; fi
Rationale:

ShellCheck found a keyword immediately followed by a [. There needs to be a space between them.

Exceptions:

None
