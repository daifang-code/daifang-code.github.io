DUO104

此linter搜索内置eval函数的使用情况。这个函数当与用户结合时，通常允许任意代码执行错误输入。

示例错误代码

malicious_user_input = 'open("/etc/passwd")'
eval(malicious_user_input)

正确代码

# Constant argument
eval('2 + 2')

来源

特别提及

如果传递给eval的数据不包含用户输入，则代码可能是安全的如果传递给eval的数据是常量字符串，则代码可能是安全的
