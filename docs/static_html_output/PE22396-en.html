PE22396: E251

Unexpected spaces around keyword / parameter equals

Bug info

BugTypeID
PE22396
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

There should be no spaces before or after the = in a function definition.

Anti-pattern
def func(key1 = 'val1',
         key2 = 'val2'):
    return key1, key2
Best practice
def func(key1='val1',
         key2='val2'):
    return key1, key2
