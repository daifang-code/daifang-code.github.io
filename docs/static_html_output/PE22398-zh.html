PE22398: E261

At least two spaces before inline comment

缺陷信息

缺陷ID
PE22398
标签
flake8
目录
Code style and efficiency
作者
pinpoint

内联注释前面应该有两个空格。程序员通常只包含一个空格，这将触发此警告。

示例错误片段
def print_name(self):
    print(self.name) # This comment needs an extra space
最佳修复
def print_name(self):
    print(self.name)  # Comment is correct now
