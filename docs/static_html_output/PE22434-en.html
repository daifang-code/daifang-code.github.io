PE22434: W293

Blank line contains whitespace

Bug info

BugTypeID
PE22434
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Blank lines should not contain any tabs or spaces.

Anti-pattern

Note: The • character represents a space.

def first_func():
    pass
••••  # This line contains four spaces

def second_func():
    pass
Best practice
def first_func():
    pass


def second_func():
    pass
