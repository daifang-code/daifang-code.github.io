PE22473: F823

Local variable name ... referenced before assignment

缺陷信息

缺陷ID
PE22473
标签
flake8
目录
Code style and efficiency
作者
pinpoint

局部变量应在引用之前定义。

示例错误片段

在本例中，func尝试递增my_varwhile还定义了一个名为my_var的新局部变量。不是的从全局命名空间引用my_var变量。

my_var = 0

def func():
    my_var += 1
最佳修复

在本例中，我们使用global语句来指示函数func应在全局命名空间。

my_var = 0

def func():
    global my_var
    my_var += 1
