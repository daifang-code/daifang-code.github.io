PE1042: 无用参数赋值

参数在函数中被修改但修改对外不可见。

缺陷信息

缺陷ID
PE1042
标签
CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

参数在函数中被修改但修改对外不可见，请检查是否这里的修改应该对外可见，也许应该传引用或者指针而不是传值。

漏洞利用威胁

低

缓解与预防

参数在函数中被修改但修改对外不可见时应该传引用或者指针而不是传值。

演示实例

漏洞代码示例
void foo(int a){ 
    a=1;
}

在上面的代码中，参数在函数中被修改但修改对外不可见采用传值。

漏洞代码修复示例
void foo(int &a){ 
    a=1; 
}

在上面的代码中，为保证修改对外可见采用传引用。
