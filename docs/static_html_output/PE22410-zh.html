PE22410: E304

Blank lines found after function decorator

缺陷信息

缺陷ID
PE22410
标签
flake8
目录
Code style and efficiency
作者
pinpoint

函数装饰器和它正在装饰的函数之间不应该有空行。

示例错误片段

在本例中，propertydecorator之间有一个空格以及name方法。空间应该被移除。

class User(object):

    @property

    def name(self):
        pass
最佳修复
class User(object):

    @property
    def name(self):
        pass
