PE0112: 浮点数的相等或不等比较

浮点数的相等或不等比较。

缺陷信息

缺陷ID
PE0112
标签
C, CPP ,FIND-BUGS
目录
Checkers/CPP
作者
pinpoint

缺陷描述

两个浮点型或双精度浮点型的值使用"=="或"!="运算符进行比较。

漏洞与风险

浮点数在计算机中的存储和计算是不精确的，因此对浮点数采用"=="或"!="运算符进行比较可能得到与期望结果相反的结果。

漏洞利用威胁

低

缓解与预防

通过计算两个浮点数之间的差的绝对值是否小于一个允许的误差值判断浮点数之间是否相等或不等。

演示实例

漏洞代码示例
bool example_fun(double a, double b) 
{
  if (a == b)  //浮点数的相等比较
  {
    return true;
  }
  return false;
}
漏洞修复代码示例
bool example_fun(double a, double b) 
{
  if (abs(a - b) < 0.000001)  //浮点数的相等比较
  {
    return true;
  }
  return false;
}
