PE22381: E128

Continuation line under-indented for visual indent

缺陷信息

缺陷ID
PE22381
标签
flake8
目录
Code style and efficiency
作者
pinpoint

为了便于视觉缩进，连续行缩进。

示例错误片段

在本例中，字符串"World"缩进两个空间。

print("Python", ("Hello",
               "World"))
最佳修复
print("Python", ("Hello",
                 "World"))
