PE22399: E262

Inline comment should start with '# '

缺陷信息

缺陷ID
PE22399
标签
flake8
目录
Code style and efficiency
作者
pinpoint

内联注释在井号（#）前应有一个空格以及评论本身。

示例错误片段
def print_name(self):
    print(self.name)  #This comment needs a space
最佳修复
def print_name(self):
    print(self.name)  # Comment is correct now
