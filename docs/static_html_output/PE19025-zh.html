PE19025: extract导致变量覆盖漏洞

extract导致变量覆盖漏洞

缺陷信息

缺陷ID
PE19025
标签
cobra
目录
Code style and efficiency
作者
pinpoint

安全风险

    变量覆盖漏洞

修复方案

    在当前行通过flags修复(eg:调用extract()时使用EXTR_SKIP保证已有变量不会被覆盖)

举例

    ```php
    <?php
        $a = '0';
        extract($_GET)；
        if($a==1){echo "true!";}
        else{echo "false!";}
    ?>
    ```
    构造 http://www.test.com/test.php?a=1时，会打印出true

匹配规则

extract
