缺陷描述

低效使用String.lastIndexOf(String)

漏洞与风险

这段代码将长度为1的常量字符串传递给String.lastIndexOf()。使用String.lastIndexOf()的整数实现会更有效。例如，调用myString.lastIndexOf('.')，而不是myString.lastIndexOf(".")
