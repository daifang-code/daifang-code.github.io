缺陷描述

类扩展了Struts Action类并使用了实例变量

漏洞与风险

此类从Struts Action类扩展，并使用实例成员变量。由于Struts框架只创建了struts Action类的一个实例，并以多线程方式使用，因此很可能存在问题，非常不鼓励这种方式。仅考虑使用方法局部变量。仅报告在监视器外部写入的实例字段。

漏洞利用威胁

高
