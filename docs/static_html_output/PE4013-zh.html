PE4013: 没有对口令域进行掩饰

用户输入口令时没有对口令域进行掩饰。

缺陷信息

缺陷ID
PE4013
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

用户输入口令时没有对口令域进行掩饰。

漏洞与风险

增加了攻击者通过观察屏幕获取到口令的可能性。

漏洞利用威胁

中

缓解与预防

要避免该漏洞，你应该：

  • 用户输入口令时对口令域进行掩饰。通常，用户输入的每一个字符都应该以星号形式回显。

演示实例

漏洞代码示例
void example_fun()
{
    char password[MAX_STR_LEN];  //MAX_STR_LEN的值为15
    int i = 0;
    char c = '0';
    printf("Please enter the password: ");
    while(i < MAX_STR_LEN && c!='\n')
    {
        scanf("%c",&c);  //没对用户输入的口令进行掩饰
        password[i] = c;
        i++;
    }
}
规范用法示例
void example_fun()
{
    char password[MAX_STR_LEN];  //MAX_STR_LEN的值为15
    int i = 0;
    char c = '0';
    printf("Please enter the password: ");
    while(i < MAX_STR_LEN && (c=getch())!='\n')
    {
        password[i] = c;
        putchar('*');  //用星号代替用户所输入的口令
        i++;
    }
}
