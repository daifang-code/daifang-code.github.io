PE22377: E123

Closing bracket does not match indentation of opening bracket's line

缺陷信息

缺陷ID
PE22377
标签
flake8
目录
Code style and efficiency
作者
pinpoint

右括号应该与左括号开始的行的缩进级别相匹配。

示例错误片段

在本例中，最后一行应与第一行缩进在同一级别。

result = function_that_takes_arguments(
    'a', 'b', 'c',
    'd', 'e', 'f',
    )
最佳修复
result = function_that_takes_arguments(
    'a', 'b', 'c',
    'd', 'e', 'f',
)
