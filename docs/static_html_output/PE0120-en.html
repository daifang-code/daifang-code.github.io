PE0120: Missing default

Missing default.

Bug info

BugTypeID
PE0120
Tags
C, CPP ,FIND-BUGS
Category
Checkers/CPP
Authors
pinpoint

Description

In the switch code block, the default logical branch is missing.

Vulnerability and risk

The readability of the code is less maintainable and can lead to a lack of logic.

Likelihood of Exploit

Low

Potential Mitigations

Make sure that each switch block has a default branch.

Demonstrative Examples

Vulnerable Code Example
void example_fun(int a)
{
  switch (a) 
  {
    case 0:
      ... 
      break;
    case 1:
      ...
      break;
  }  //Missing default
}
Fixed Code Example
void example_fun(int a)
{
  switch (a) 
  {
    case 0:
      ... 
      break;
    case 1:
      ...
      break;
    default:
      ...
  }  
}
