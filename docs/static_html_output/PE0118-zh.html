PE0118: 对非函数使用inline

在非函数或方法中使用了"inline"关键字。

缺陷信息

缺陷ID
PE0118
标签
C, CPP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

在非函数或方法中使用了"inline"关键字。

漏洞与风险

"inline"关键字是用于定义内联函数的，定义变量无意义。

漏洞利用威胁

低

缓解与预防

不应在非函数或方法中使用"inline"关键字。

演示实例

漏洞代码示例
inline int a;  // 在定义变量中使用了"inline"关键字
