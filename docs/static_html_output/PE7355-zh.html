缺陷描述

方法使用零长度数组作为toArray()的参数

漏洞与风险

该方法使用派生类集合的toArray()方法，并传递到一个长度为零的原型数组参数中。 这种情况下使用myCollection.toArray(new Foo[myCollection.size()])更有效，因为如果传入的数组足够大以至于可以存储集合中的所有元素，那么它将被填充并直接返回。这就避免了创建第二个数组(通过反射)来返回结果的需求。

漏洞利用威胁

低
