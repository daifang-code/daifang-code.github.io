PE1076: 资源注入

攻击者通过操纵资源标识符来利用验证输入信息过程中的缺陷

缺陷信息

缺陷ID
PE1076
标签
JAVA
目录
Checkers/JAVA
作者
pinpoint

缺陷描述

攻击者通过操纵资源标识符来利用验证输入信息过程中的缺陷，从而意想不到地实现某个资源的修改或说明（modification or specification）。

漏洞与风险

某个资源有个能遭受无意的修改或说明。

漏洞利用威胁

中

缓解与预防

  • 确保所有发送给客户端的输入内容都经过了一个可接受的内容说明的清洗。
  • 对所有的内容进行输入验证。
  • 强制对软件定期打补丁。
