PE22408: E302

Expected 2 blank lines, found 0

缺陷信息

缺陷ID
PE22408
标签
flake8
目录
Code style and efficiency
作者
pinpoint

函数和类之间应有两个空行。

示例错误片段
def func1():
    pass
def func2():
    pass
最佳修复
def func1():
    pass


def func2():
    pass
