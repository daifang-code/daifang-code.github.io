缺陷描述

方法调用低效的new String()构造函数会浪费内存空间

漏洞与风险

此方法使用没有参数的构造方法去创建新的String对象，这样做会浪费内存空间。因为这样创建对象会和空字符串""效果相同。Java中保证完成相同的构造方法会产生描述相同的String对象。所以你只要使用空字符串来创建就可以了。
