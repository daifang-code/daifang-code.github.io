缺陷描述

不要使用removeAll来清除一个collection

漏洞与风险

如果要从集合c中删除所有元素，请使用c.clear，而不是c.removeAll(c)。调用c.removeAll(c)清除集合意义不是很明确，容易受到拼写错误的影响，效率较低，对于某些集合，可能会抛出ConcurrentModificationException。

低
