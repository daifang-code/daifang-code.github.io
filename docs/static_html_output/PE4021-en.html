PE4021: uncaught except

uncaught except

Bug info

BugTypeID
PE4021
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

Programs will terminate abnormally when an exception is thrown but never caught or the exception does not conform to the exception specifier of the function.

Likelihood of Exploit

High

Demonstrative Examples

void no_throw() throw(){  // detect
    throw(12); 
}


void except_true() noexcept(true){ // detect
    throw(12);  
}

int main(int argc, char **argv) { // detect
    throw(1);
    return 0;
}
