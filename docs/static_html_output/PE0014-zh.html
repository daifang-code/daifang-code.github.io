PE0014: 函数缺少返回值

返回非void类型的函数在某条路径上缺少返回值。

缺陷信息

缺陷ID
PE0014
标签
C, CPP, JAVA, CSHARP
目录
Checkers/CPP
作者
pinpoint

缺陷描述

返回非void类型的函数在某条路径上缺少返回值，会导致出乎意料的程序行为。

漏洞与风险

出乎意料的行为

返回非void类型的函数在某条路径上缺少返回值，会导致出乎意料的程序行为。

漏洞利用威胁

中

缓解与预防

在返回非void类型的函数的每条路径上都提供返回值。

演示实例

漏洞代码示例
int bad(int x) {
  if (x == 5)
    return 4;
  else if (x == 3)
    return 2;
} // missing_return

在上面的代码中，当x的值既不等于5，也不等于3时，函数bad缺少返回值。

漏洞代码修复示例
int good(int x) {
  if (x == 5)
    return 4;
  else if (x == 3)
    return 2;
  return 0;
}

在上面的代码中，在函数的最后提供了返回值0。
