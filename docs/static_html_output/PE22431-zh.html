PE22431: W191

Indentation contains tabs

缺陷信息

缺陷ID
PE22431
标签
flake8
目录
Code style and efficiency
作者
pinpoint

只有空格时缩进有制表符。将所有选项卡更改为空格。

示例错误片段

注意：→代表一个选项卡。

def get_name(self):
    if self.first_name and self.last_name:
    → → return self.first_name + ' ' + self.last_name
    else:
        return self.last_name
最佳修复

仅使用空格。

def get_name(self):
    if self.first_name and self.last_name:
        return self.first_name + ' ' + self.last_name
    else:
        return self.last_name
