PE22420: E704

Multiple statements on one line (def)

Bug info

BugTypeID
PE22420
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Multiple statements of a function definition should be on their own separate lines.

Anti-pattern
def f(): pass
Best practice
def f():
    pass
