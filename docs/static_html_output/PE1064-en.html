PE1064: Incompleted initialization

Bug info

BugTypeID
PE1064
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

Incorrect use of the memset, memcpy or memmove functions causes the buffer fail to be fully evaluated.

Vulnerability and risk

Incorrect Calculation of Buffer Size

Incorrect use of the memset, memcpy or memmove functions causes the buffer fail to be fully evaluated.

Likelihood of Exploit

Use the memset, memcpy, or memmove functions correctly.

Potential Mitigations

medium

Demonstrative Examples

Vulnerable Code Example
void bad()
{
  int a[5];
  memset(a, 0, 5);//detect
}
Fixed Code Example
void good()
{
  int a[5];
  memset(a, 0, 5*sizeof(int));
}
