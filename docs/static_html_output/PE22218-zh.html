DUO108

此linter搜索内置input函数的使用情况。这个函数相当于eval（raw_input（…）），因此通常允许任意与用户输入结合时，代码执行错误。

示例错误代码

input('Text here: ')
Text here: open('/etc/passwd').read()

正确代码

raw_input('Text here: ')
Text here: open('/etc/passwd').read()

来源

特别提及

*代码使用六步移动输入其行为类似于python3的input，因此是安全的
