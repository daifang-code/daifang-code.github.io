PE22400: E265

Block comment should start with '# '

缺陷信息

缺陷ID
PE22400
标签
flake8
目录
Code style and efficiency
作者
pinpoint

块注释在井号（#）前应有一个空格以及评论本身。

示例错误片段
#This comment needs a space
def print_name(self):
    print(self.name)
最佳修复
# Comment is correct now
def print_name(self):
    print(self.name)
