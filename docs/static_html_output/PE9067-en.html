PE9067: Redundant Header Includes

Find and optionally remove unneeded includes in C or C++ source files.

Bug info

BugTypeID
PE9067
Tags
C, CPP
Category
Checkers/CPP
Authors
pinpoint

Description

This checker analyzes C and C++ files to determine which header inclusions can be removed while still allowing them to compile. This may result in substantial improvements in compilation time, especially on large C++ projects; it also sometimes exposes dependencies and cohesions of which developers were unaware.

Vulnerability and risk

The appearances of redundant headers cause:

  • Compilation slowdown
  • Redundant Recompilations
  • Disallow refactoring
  • Make a project not self-documented

Likelihood of Exploit

Medium

Potential Mitigations

Remove the redundant header files from the source code.

Demonstrative Examples

Vulnerable Code Example
#include <iostream>  // Redundant Header Includes

int test() {
    return 1 + 2;
}

In the above code, not a symbol included in <iostream> is used. Thus, it is redundant and should be removed.
