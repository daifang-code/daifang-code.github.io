PE22417: E701

Multiple statements on one line (colon)

Bug info

BugTypeID
PE22417
Tags
flake8
Category
Code style and efficiency
Authors
pinpoint

Multiple statements should be on their own separate lines. This improves readability.

Anti-pattern
if x > 5: y = 10
Best practice
if x > 5:
    y = 10
