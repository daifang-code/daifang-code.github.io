PE22440: W602

Deprecated form of raising exception

缺陷信息

缺陷ID
PE22440
标签
flake8
目录
Code style and efficiency
作者
pinpoint

raise Exception, message引发异常的形式是已弃用。使用新表单。

示例错误片段
def can_drive(age):
    if age < 16:
        raise ValueError, 'Not old enough to drive'
    return True
最佳修复
def can_drive(age):
    if age < 16:
        raise ValueError('Not old enough to drive')
    return True
