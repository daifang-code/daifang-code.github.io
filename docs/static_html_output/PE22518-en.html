Message

Explicit return in init

Description

Used when the special class method init() has an explicit return value. This message belongs to the basic checker.

Explanation

The init() method is required to return nothing. Python 2.7 and 3.x raises:

TypeError: init() should return None

when init() is called and executes a return statement with a value other than None. PyLint reports this error without depending on the actual invocation.
