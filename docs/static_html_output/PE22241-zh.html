DUO131

此linter搜索urllib3模块的不安全属性使用。特别是使用disable_warnings函数。此功能禁用警告，如未验证的HTTPS请求、没有SNI的HTTPS请求可用，对证书丢失SAN的主机的HTTPS请求，等等。

示例错误代码

import urllib3
urllib3.disable_warnings()

来源

urllib3生成的警告旨在阻止用户执行不安全的web请求（同时防止其他常见的，非安全相关错误）。禁用此功能会导致不安全不被注意的请求。
