PE0109: 对BSTR类型的变量进行地址运算

对BSTR类型的变量进行地址运算。

缺陷信息

缺陷描述

对BSTR类型的变量进行地址运算。

漏洞与风险

对BSTR类型的变量进行地址运算可能导致错误的内存访问。

漏洞利用威胁

低

缓解与预防

不对BSTR类型的变量进行地址运算。

演示实例

漏洞代码示例
void example_fun(BSTR a) 
{
    a++;  //对BSTR类型的变量进行地址运算
    ...
}
