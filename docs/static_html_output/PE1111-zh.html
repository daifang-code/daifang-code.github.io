PE1111: 劫持Android类加载

攻击者能够通过改变命令的执行环境，进而使程序从非受信的资源中加载库

缺陷信息

缺陷ID

标签
JAVA

缺陷描述

从非受信的资源中或在非受信的环境下执行命令或加载库都可能会使得程序以攻击者的名义执行任意的恶意代码。

漏洞与风险

攻击者能够通过改变命令的执行环境，进而使程序从非受信的资源中加载库。

漏洞利用威胁

中

缓解与预防

要清楚了解待加载的库，并从可信源进行加载。
