缺陷描述

抽象类定义了covariant compareTo()方法

漏洞与风险

这个类定义了一个协变的 compareTo()；

若要正确地重写<code>Comparable</code> 接口中的<code>compareTo()</code>方法， <code>compareTo()</code>的参数必须是<code>java.lang.Object</code>类型。</p>
