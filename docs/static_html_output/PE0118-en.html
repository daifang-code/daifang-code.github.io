Description

Inline variable declaration.

Vulnerability and risk

The "inline" keyword is used to define inline functions, and defining variables is meaningless.

Likelihood of Exploit

Low

Potential Mitigations

The "inline" keyword should not be used in non-functions or methods.

Demonstrative Examples

Vulnerable Code Example
inline int a;  // The "inline" keyword is used in defining variables
