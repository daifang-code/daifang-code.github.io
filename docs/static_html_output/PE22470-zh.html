PE22470: F812

List comprehension redefines name from line n

缺陷信息

缺陷ID
PE22470
标签
flake8
目录
Code style and efficiency
作者
pinpoint

列表理解使用与模块中定义的另一个变量相同的名称。更改列表理解中的变量名称或在模块中更改它。

示例错误片段

在本例中，变量i被定义为1，然后在列表理解。

i = 1
squares = [i ** 2 for i in range(10)]
最佳修复
i = 1
squares = [num ** 2 for num in range(10)]
