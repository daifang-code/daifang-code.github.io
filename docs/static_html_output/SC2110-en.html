SC2110: Suspicious condition

In [\[..]], use || instead of -o.

In [[..]], use || instead of -o.

Problematic code:
[[ "$1" = "-v" -o "$1" = "-help" ]]
Correct code:
[[ "$1" = "-v" || "$1" = "-help" ]]
Rationale:

-o for logical OR is not supported in a [[ .. ]] expression. Use || instead.

Exceptions:

None.
